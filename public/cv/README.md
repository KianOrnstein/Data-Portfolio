# CV Folder

## Cách sử dụng

Đặt file CV PDF của bạn vào thư mục này với một trong các tên sau:

- `cv.pdf` (khuyến nghị)
- `resume.pdf`
- `CV.pdf`
- `Resume.pdf`
- `curriculum-vitae.pdf`

Hoặc bất kỳ file PDF nào - hệ thống sẽ tự động tìm file PDF đầu tiên trong thư mục này.

## IELTS Certificate

Đặt file IELTS certificate PDF vào thư mục này với một trong các tên sau:

- `ielts.pdf` (khuyến nghị)
- `IELTS.pdf`
- `ielts-certificate.pdf`
- `IELTS-certificate.pdf`

## Lưu ý

- Chỉ đặt **1 file CV PDF** trong thư mục này
- File IELTS có thể đặt cùng folder với CV
- File sẽ được hiển thị tự động trên trang web
- Bạn có thể download file trực tiếp từ trang web
